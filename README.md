# VITREE Content Hub

Public educational content database for VITREE Health Technologies apps:

- LUNARA
- HERBORA
- VITREE

## Purpose

This repository stores public educational JSON content for mobile apps.

## Important safety rules

This repository must never contain:

- API keys
- `.env` files
- Gemini API keys
- user data
- pregnancy notes
- medical reports
- private health data
- login data
- passwords

## Content only

This repo is for public educational app content only:
- documents
- medication safety content
- doctor questions
- red flags
- herbs
- exercises
- meditations

## Apps

LUNARA: pregnancy, documents, medication safety, doctor questions  
HERBORA: herbs and plant safety  
VITREE: wellness, mind/body/spirit growth content
